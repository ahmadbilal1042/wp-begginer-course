#permalinks

Dynamic urls of pages

possible to display different types of content using same directory file like pages, posts 

In WordPress core, a permalink means the permanent URL of a post, page, or custom content on your website.


#query strings

any specific string in url passed as parameter and sent to server to request specific data


#permalinks

permalinks use rules rewrite the query strings to more readable format


URL mapping  (url rewriting)

URL rewriting is configured in .htaccess file in apache server

in ngnix it is configured in a server block as htaccess is restricted in ngnix
